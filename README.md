
 # Smart Attendance System (QR Based)


## Description
This project is a QR-based smart attendance system where students scan a dynamic QR code to mark attendance.

## Technologies Used
- Flask (Backend)
- HTML, CSS, JavaScript (Frontend)
- QR Code Generation (Python qrcode)
- ngrok (for public access)

## Features
- Time-based QR code generation
- Student attendance marking
- Simple login system
- Attendance record display

## How to Run
1. Run Flask app:
   python app.py

2. Start ngrok:
   ngrok http 5000

3. Copy ngrok URL and update in app.py

4. Scan QR to mark attendance
